◆ criar_objeto_de_texto()

Objeto * criar_objeto_de_texto ( int espacamento,
int tam_fonte,
const char * txt_formatado,
... )

Cria um objeto de texto a partir de um texto.

Parâmetros
espacamentoDistância (em pixel) entre cada caractere
tam_fonteO tamanho da fonte, podendo ser 1, 2 ou 3. Quanto maior a fonte, maior a resolução.
txt_formatadoO texto formatado a ser convertido no objeto de texto
Nota
Apenas 40 caracteres são suportados. Letras de A - Z; Algarismos de 0 - 9; e os caracteres especiais: Ponto - . Vírgula - , Exclamação ! Interrogação ?

Vector2 pos,

... )

Realiza printf para escrever um texto no terminal dada uma cor e posição.

Parâmetros
cor_do_textoA cor do texto que será printado
posA posição do texto na tela, 1-indexado com referencial de matriz. Note que (-1, -1) não alterará a posição do cursor
formatO texto formatado a ser printado
Nota
Para fazer o print no mesmo lugar onde o cursor já está, passe o vetor como (-1, -1)
